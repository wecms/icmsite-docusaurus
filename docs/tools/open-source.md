### 搭载全志H616的核桃派1B再度刷屏朋友圈

树莓派一直以极高的性价比和生态著称，然而在过去两年全球缺芯的大环境下，大部分用户被迫使用“高价派”。

价格一去不复返的树莓派给了国产替代开发板很多上位的机会，像与树莓派ZeroW板型相仿的芒果派MQ Pro、香橙派Zero2W等，也有像与树莓派4B板型相仿的BIGTREETECH CB1，诸如此类的国产派如雨后春笋般涌现。
